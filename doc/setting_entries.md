Einträge für die Tabelle `sddl$dict_setting`
============================================

Die Einträge können von jedem Entwickler beliebig verändert werden.

### SECTION: `PREFIX`

* IDENT: `VIEW`              
  VALUE: `V_`

* IDENT: `TABLE`              
  VALUE: [leerstring]

* IDENT: `SEQUENCE`              
  VALUE: `SEQ_`

* IDENT: `STOREDPROCEDURE`              
  VALUE: `SP_` 

* IDENT: `FOREIGNKEY`              
  VALUE: `FK_`

* IDENT: `INDEX`              
  VALUE: `IDX_`

* IDENT: `UNIQUEKEY`              
  VALUE: `UNQ_`

* IDENT: `M:N`              
  VALUE: `REL_`

* IDENT: `CATALOG`              
  VALUE: `CAT_`
  
### SECTION: `SCRIPT.COMMENT`
Notwendige Syntaxinformationen zur Kommentierung in Scripten erstellt.

* IDENT: `START`              
  VALUE: `/*`

* IDENT: `END`              
  VALUE: `*/`

* IDENT: `PAD`              
  VALUE: `-`

* IDENT: `LEN`              
  VALUE: `100`
  
### SECTION: `SCRIPT.HEADER`
Notwendige Informationen für einen Header in einem Script.

* IDENT: `ACTIVE`              
  VALUE: [0|1] 0=kein Header  

* IDENT: `AUTHOR`              
  VALUE: `<Vorname> <Name>`

* IDENT: `TITEL`              
  VALUE: [entsprechend]

* IDENT: `COPYRIGHT`              
  VALUE: [entsprechend]   
  
* IDENT: `COMMENT`              
  VALUE: [entsprechend]
